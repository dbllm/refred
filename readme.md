The project structure is as follows:

```
cllm/
    aux/ -- tools for obtaining LLM responses, embeddings, etc.
    reg/ -- the main code for experiments.
        paper_main.py -- the main code for producing the results in the paper.
        paper_figure.py -- the code for producing the figures in the paper.
    dataset/ -- code for producing the datasets.
setup.py -- the setup file for the project.
datasets/ -- the datasets folder. Downloaded from Google Drive. https://drive.google.com/file/d/1JiugznC3-Lql-WcGwwU_DKUSLOrqhGqy/view?usp=sharing
readme.md -- this file. 
```
